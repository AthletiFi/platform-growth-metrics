# Pleyr Platform Growth Metrics

> Real-time analytics and growth metrics from Pleyr's platform launch

## About Pleyr

Pleyr is a gamified, video-centric social platform designed to give youth athletes ownership of their athletic journey. The platform solves the "broken pipeline" problem by providing:

- **Team Video Vault** - Centralized storage where parents, coaches, and teammates upload full games
- **Digital Player Cards** - Shareable, trading-card-style profiles showcasing athlete achievements
- **Athlete-First Feed** - A social space where only players can post publicly
- **Game Log** - Post-game reflections that showcase character and resilience

## Dashboard Overview

This dashboard visualizes key metrics from Pleyr's initial launch period (October 15 - November 7, 2025), showcasing:

- **30 total users** in first 23 days
- **100% profile completion rate** - demonstrating strong product-market fit
- **70% card creation rate** - validating the core digital trading card concept
- **26 videos uploaded** - athletes accessing their own game footage
- **14 unique teams** across multiple states and international locations
